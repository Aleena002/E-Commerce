# E-Shop React Application

## Project Overview
E-Shop is a simple e-commerce web application built with React and Bootstrap.  
It lets users browse products, view details, add items to the cart, and includes an admin dashboard for managing users and orders.

---

## Pages and Components Implemented

### Pages
- **Home Page** (`/`) – Shows a hero banner and a “Shop Now” button  
- **Products Page** (`/products`) – Displays a list of featured products with Add to Cart option  
- **Product Details Page** (`/product/:id`) – Shows details of a single product and allows adding it to the cart  
- **Cart Page** (`/cart`) – Displays all products in the cart with quantity and total price  
- **Login Page** (`/login`) – Admin login form  
- **Admin Dashboard** (`/admin`) – Shows total users, orders, revenue, and a users table

### Components
- **Navbar** – Navigation links to different pages  
- **Footer** – Footer section  
- **ProductCard** – Shows product image, name, price, and buttons for details and cart

## Steps to Run the Project

1. Clone the repository:

git clone <repository-link>


2. Go to the project folder:

cd project-folder


3. Install dependencies:

npm install

4. Start the development server:

npm start


Then open [http://localhost:3000](http://localhost:3000) in your browser to see the app.
